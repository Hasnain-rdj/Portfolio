Software Quality Testing

Software Quality Testing (SQT) is a critical process in the software development lifecycle (SDLC) that ensures the software meets the required quality standards and functions as intended. The objective of software testing is to identify defects, ensure quality assurance, and provide the confidence that the software is reliable and performs correctly. SQT is typically divided into several key phases, each with distinct objectives.

Web-Based Calculator

I built a simple web-based calculator application using HTML, CSS, and JavaScript. The HTML structure defines the layout of the calculator, including buttons for digits, operations, and the display screen. Using CSS, I styled the calculator to ensure it has a clean and user-friendly interface with proper button sizes, margins, and a sleek design. JavaScript powers the functionality by handling user input, performing arithmetic operations (addition, subtraction, multiplication, and division), and updating the display dynamically. The calculator uses event listeners to capture button clicks and executes calculations, offering a smooth and interactive user experience.

Railway Management System

The Railway Management System I developed utilizes C# for the backend and MySQL Server to manage and store passenger data. The system allows real-time tracking of trains, providing updates on train locations, arrival times, and delays. Passengers' details, such as booking information, seat availability, and travel history, are stored securely in the MySQL database. The C# application interfaces with the database to add, update, and retrieve passenger records, while also handling train schedules and ticketing operations. The system integrates live data for train tracking, ensuring users can view train statuses in real-time, enhancing the overall experience for both passengers and railway staff.
